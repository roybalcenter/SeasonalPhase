# SeasonalPhase
Repository for ESSR paper on seasonal phases

Contents:
Alternate Figure 1 (oriented around seasons in the southern hemisphere)
Sample Python code for changing dates into astronomical seasonal phase
Sample R script for changing dates into astronomical seasonal phase
Sample SPSS syntax for changing dates into astronomical seasonal phase
Sample SAS program for changing dates into astronomical seasonal phase

Summary of standardized variables:
date_var: The input date variable.
day_of_year: The numeric day (1–366).
seasonal_phase: The resulting continuous wave variable (+1 to -1).

All implementations use the variable seasonal_phase to represent the continuous wave output.
